PROJECT STRUCTURE:
─────────────────────────────────────────────────────────────────────────────

Digital-Skills-Framework---Weston/
│
├── 📄 index.html                           [MAIN APPLICATION]
│   • 8-tab HTML5 app with Playfair Display + Lato fonts
│   • Passcode modal (2402) for secure editing
│   • Tabs 2, 3, 5, 8 are editable (with unlock)
│   • Tab 8: NEW — Digital Leads table (Area Code, Name, HOA, Digital Lead)
│   • Fetches areas-data.json from GitHub on page load
│   • Calls Netlify Edge Function (/.netlify/functions/sync-data) to save edits
│   • WCAG 2.2 AA compliant (colour contrast verified)
│   • ~1500 lines | 27 KB
│
├── 📊 areas-data.json                      [DATA SOURCE OF TRUTH]
│   • Single JSON file with all editable content
│   • Structure: digitalLeads[], areas[], industrySkills{}, sendProvisions{}
│   • Synced via Netlify Edge Function when you edit
│   • Fetched by browser every 1–2 seconds to ensure live updates
│   • Version: 3.0 | Last updated: 2026-04-28
│   • ~300 lines | 7 KB
│
├── ⚙️ netlify.toml                         [DEPLOYMENT CONFIG]
│   • Tells Netlify where Edge Functions are
│   • Configures static site serving
│   • Sets environment variable requirements (GITHUB_PAT)
│   • CORS headers for cross-origin requests
│   • ~25 lines | 1 KB
│
└── 📁 netlify/edge-functions/
    │
    └── sync-data.ts                        [SERVER-SIDE PROXY - CRITICAL]
        • TypeScript Edge Function (runs on Netlify CDN)
        • Receives edit requests from browser
        • **HOLDS YOUR GITHUB PAT** (never exposed to browser)
        • Validates incoming data
        • Commits updated areas-data.json to GitHub via GitHub API
        • Returns success/error response to browser
        • Auto-includes timestamps & commit metadata
        • ~200 lines | 7 KB

KEY FEATURES:
─────────────────────────────────────────────────────────────────────────────

✅ 8 Tabs:
   • Tab 1: General Digital Skills (read-only reference)
   • Tab 2: All Subjects & Areas (✏️ EDITABLE)
   • Tab 3: Industry-Specific Skills (✏️ EDITABLE)
   • Tab 4: Framing for Industry (read-only reference)
   • Tab 5: SEND, Inclusion & Accessibility (✏️ EDITABLE)
   • Tab 6: Skills Progression (read-only reference)
   • Tab 7: Teacher Digital Skills (read-only reference)
   • Tab 8: Digital Leads by Area (✏️ EDITABLE — NEW!)

✅ Passcode Protection:
   • 4-digit passcode (2402) required to unlock editing
   • Validated in browser (not sent to server)
   • Prevents accidental edits
   • Only you know the code

✅ GitHub Sync:
   • All edits write to areas-data.json in GitHub
   • All users see changes within 1–2 seconds
   • GitHub keeps complete version history
   • Netlify Edge Function handles the proxy securely

✅ Security:
   • GitHub PAT never exposed to browser
   • Stored only in Netlify environment variables
   • All commits timestamped and logged
   • HTTPS/CORS protected

✅ Accessibility:
   • WCAG 2.2 AA compliant
   • All colour contrasts verified:
     - White on Navy: 16.59:1 (AAA level)
     - Teal on Navy: 6.66:1 (AA level)
     - Text on Cream: 17.95:1 (AAA level)
   • Keyboard accessible, focus outlines visible
   • Form labels properly associated
